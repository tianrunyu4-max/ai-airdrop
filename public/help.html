💻 电脑端问题解决

问题：登录失败或页面加载错误

  1. 访问：https://eth10.netlify.app/fix-now.html
  2. 点击"立即修复登录问题"
  3. 等待自动清理缓存
  4. 跳转到登录页面后重新登录

快捷键清理缓存

  • Windows/Linux: 按 Ctrl + Shift + Delete
  • Mac: 按 Command + Shift + Delete
  • 选择"全部时间"
  • 勾选"缓存的图片和文件"
  • 点击"清除数据"

🔧 手动清理浏览器缓存

iPhone Safari

  1. 打开"设置"应用
  2. 向下滚动找到"Safari"
  3. 点击"清除历史记录与网站数据"
  4. 确认清除
  5. 重新打开Safari访问网站

Android Chrome

  1. 打开Chrome浏览器
  2. 点击右上角三个点
  3. 选择"设置" → "隐私"
  4. 点击"清除浏览数据"
  5. 选择"全部时间"
  6. 勾选"缓存的图片和文件"
  7. 点击"清除数据"

微信浏览器

  1. 在网页右上角点击"..."
  2. 选择"设置"
  3. 点击"清理缓存"
  4. 确认清理
  5. 关闭后重新打开网页

🌐 网络问题排查

  • 检查网络连接：确保手机/电脑已连接到互联网
  • 切换网络：尝试从WiFi切换到4G/5G，或反之
  • 关闭VPN：如果使用了VPN或代理，尝试关闭
  • 重启设备：重启手机或电脑后再试
  • 更换浏览器：尝试使用其他浏览器（Chrome、Safari、Firefox）
  • 使用无痕模式：在浏览器的无痕/隐私模式下访问
