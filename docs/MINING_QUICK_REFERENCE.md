## 🔑 核心公式

```
每日释放 = 投入 × (1% + 加速率)
加速率 = min(直推数 × 1.5%, 10%)
出局天数 = 总产出 ÷ 每日释放
兑换U = 总产出 × 0.07 × 0.7
互转积分 = 总产出 × 0.3
```

## 📝 快速计算器

### 计算日均收益

```
日均U = (投入 × 倍数 × 0.07 × 0.7) ÷ 出局天数
```

**示例**:
```
一型: (100 × 10 × 0.07 × 0.7) ÷ 91 = 0.54U/天
二型: (1000 × 2 × 0.07 × 0.7) ÷ 19 = 5.16U/天
三型: (5000 × 2 × 0.07 × 0.7) ÷ 19 = 25.79U/天
```
