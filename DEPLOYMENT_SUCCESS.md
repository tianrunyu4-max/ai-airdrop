## 🔧 **配置Supabase（可选）：**

### **如果你想启用完整功能：**

**第1步：在Netlify添加环境变量**

访问：https://app.netlify.com/projects/eth10/settings/env

添加：
```
VITE_SUPABASE_URL=你的Supabase URL
VITE_SUPABASE_ANON_KEY=你的Supabase密钥
```

**第2步：重新部署**

```powershell
netlify deploy --prod --dir=dist
```

**第3步：验证**

打开网站，尝试注册，如果成功说明Supabase已连接！
